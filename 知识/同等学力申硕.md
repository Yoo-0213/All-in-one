# 介绍

**官方解释**

同等学力是指未取得国家承认的相应层次的学历，而具有同层次学力水平，初步认定为同等学力

同等学力申硕是具有研究生毕业同等学力的人员，可按照《国务院学位委员会关于授予具有研究生毕业同等学力人员硕士、博士学位的规定》的具体要求与办法，向学位授予单位提出学位申请，申请人通过国家组织的全部考试以及学位论文答辩后，可以获得国家承认的硕士学位

**总结**

同等学力申硕可以理解成**在职人员获取硕士学位的一种方式**，由于已经出社会工作的原因，再花几年时间去参加全国硕士研究生招生并不现实，而同等学力申硕仅仅是**为了获取硕士学位**，即**证明我在学术方面已经达到硕士学位的学术水平**，这个貌似企业的HR和领导对这个的认可度还行？

但是目前来看并没有发现对个人有多少好处

**区别**

| **身份**  | **时间分配**             | **学位证** | **毕业证** |
| ------- | -------------------- | ------- | ------- |
| 全日制研究生  | 全部时间在校学习             | ✔️      | ✔️      |
| 非全日制研究生 | 部分时间在工作，部分时间**在校**学习 | ✔️      | ✔️      |
| 同等学力申硕  | 部分时间在工作，业余时间**网上**学习 | ✔️      | ❌       |

# 考生资格

---

- **已获取学士学位，并在获取学士学位后工作3年**（学士学位即我们的本科学位）；虽无学士学位，但已获得硕士或博士学位者；通过教育部留学服务中心认证的国（境）外学士、硕士或博士学位获得者

<aside> 💡

**学力**

- 一个人掌握知识和技能的能力，通常反映一个人在某一领域的学术水平、学习能力、思考能力等

**学位**

- 学位是标志一个人学术水平达到一定标准的学术称号
- 在中国大陆，学位分为三个级别：学士、硕士和博士
- 获得学位需要满足一定的学习成绩、课程要求和学术水平

**学历**

- 指一个人在学校学习的经历，有学历不一定有学位

大专：有毕业证，没有学位证 本科：有毕业证，也有学位证

</aside>
